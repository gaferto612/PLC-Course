# Sprint 4 — The Other Four IEC 61131-3 Languages

> **Bolton chapter:** 6
> **Time budget:** 6–8 hours
> **Core concept:** IEC 61131-3 standardizes five interchangeable languages so you pick the right tool for each part of the program — text for math, ladder for interlocks, SFC for sequences.

---

## 🎯 Sprint Goals

- Read and write basic Instruction List (IL), recognize when to leave it for legacy code
- Write Structured Text (ST) for math, conditions, and loops
- Build a Sequential Function Chart (SFC) for a multi-step batch process
- Use the language decision tree to pick the right language for a sub-task
- Map each IEC 61131-3 language to its Siemens TIA Portal equivalent (KOP / FUP / SCL / GRAPH / AWL)
