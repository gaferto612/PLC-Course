# Workbooks

This folder will hold Jupyter notebooks for hands-on exercises tied to each sprint:

- `01-io-mapping.ipynb` — classify 10 real-world devices as discrete/analog and PNP/NPN
- `02-number-systems.ipynb` — drills and 2's complement implementation
- `03-ladder-translation.ipynb` — written specs → ladder + FBD side-by-side
- `04-st-vs-il.ipynb` — translate-the-rung exercises
- `05-timer-state-machines.ipynb` — implement TON/TOF/TP behavior in Python
- `06-safety-fmea.ipynb` — FMEA template + analysis exercise

To get started, install jupyter and create your own notebook for each sprint:

```bash
pip install jupyter
jupyter notebook
```

These workbooks are intentionally not pre-populated — the act of building them yourself is the learning. Each sprint README has a brief outline of what its workbook should cover.

If you'd like to contribute polished workbooks back, see [CONTRIBUTING.md](../CONTRIBUTING.md).
